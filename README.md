# HTML PPT Generator

一个用于创建交互式HTML演示文稿的Claude Code技能。只需提供主题、配色方案和内容，即可生成专业美观的PPT。

[English](./README_EN.md) | 中文

## 特性

- **多套预设配色** - 赛博朋克、商务蓝、暗夜紫、森林绿等8套方案
- **自定义配色** - 支持完全自定义颜色配置
- **丰富组件** - 标题页、卡片列表、代码块、时间线、统计卡片等
- **交互功能** - 键盘导航、触摸滑动、缩略图视图、全屏模式
- **动画效果** - 粒子浮动、毛玻璃、弹性入场动画
- **零依赖** - 纯本地化方案，无需任何外部CDN

## 快速开始

### 1. 安装技能

将 `html-ppt` 技能文件夹复制到你的 Claude Code 技能目录：

```
.claude/skills/html-ppt/
├── SKILL.md                 # 技能说明
├── README.md                # 本文档
├── templates/
│   ├── base_template.html   # 基础模板
│   └── color_schemes.json   # 配色配置
└── examples/                # 示例文件（可选）
```

### 2. 使用方法

在Claude Code中调用此技能：

```
/html-ppt
```

然后按提示提供：
1. **配色方案** - 从预设选择或自定义
2. **主题** - PPT标题
3. **每页内容** - 标题、内容类型、具体内容

### 3. 输出

生成的PPT是一个独立的 `.html` 文件，可直接在浏览器中打开。

### 自定义配色

你也可以指定完整颜色配置：

```json
{
  "bg-deep": "#0a0a0f",
  "bg-card": "rgba(20, 20, 30, 0.85)",
  "primary": "#00f5d4",
  "secondary": "#7b2cbf",
  "accent": "#ff6b35",
  "text-primary": "#ffffff",
  "text-secondary": "#a0a0b0",
  "border-glow": "rgba(0, 245, 212, 0.3)"
}
```

## 快捷键

| 按键 | 功能 |
|------|------|
| → / 空格 / 滚轮向下 | 下一页 |
| ← / 滚轮向上 | 上一页 |
| T | 切换缩略图视图 |
| F | 切换全屏模式 |
| H | 显示帮助 |
| ESC | 退出全屏/缩略图 |

## 可用组件

1. **标题幻灯片** - 大标题 + 副标题 + 动态背景
2. **内容幻灯片** - 标题 + 文字内容
3. **卡片列表** - 多列卡片展示特性
4. **代码块** - 带语法高亮的代码展示
5. **图片展示** - 图片/截图展示区域
6. **对比卡片** - 左右对比布局
7. **特性列表** - 带图标的特性项
8. **引用框** - 案例/引用展示
9. **总结幻灯片** - 总结内容 + 联系方式
10. **时间线** - 流程/步骤展示
11. **统计卡片** - 数字/统计展示

## 示例输出

## 技术限制

- **零外部依赖** - 禁止引入任何CDN链接
- **系统字体** - 使用系统预装字体
- **单文件输出** - 所有CSS/JS内联到HTML中
- **无滚动条** - 每页内容必须完全在可视区域内
